#  Marker-Based Furniture AR Application (Unity + Vuforia)

##  Project Overview

This application uses Vuforia Image Targets to visualize furniture models in Augmented Reality.
When the camera detects a marker image, the corresponding 3D furniture model appears along with information text.

This project demonstrates marker-based tracking and AR product visualization.

---

##  Features

* Image target detection
* Multiple markers support
* 3D furniture models
* Real-time tracking
* Text labels (name, price, details)
* UI canvas for information
* Interactive AR experience

---

##  Technologies Used

* Unity
* Vuforia Engine SDK
* C#
* TextMeshPro
* Android platform

---

##  Project Structure

Assets/

* ImageTargets Database
* Furniture Models
* Prefabs
* Scripts
* UI
* ARCamera

---

## ▶How to Run

1. Open project in Unity
2. Import Vuforia Engine
3. Add License Key in Vuforia Configuration
4. Activate Image Target Database
5. Build for Android
6. Print markers and scan with camera

---

##  Learning Outcomes

* Marker-based AR concepts
* Image target creation
* Model placement on targets
* UI in world space
* AR product visualization techniques

---

## Usage

Scan different markers:

* Chair → Chair model
* Table → Table model
* Sofa → Sofa model

Each marker displays separate furniture.
